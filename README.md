# Instalar as dependências

## Node

Versão node: 18.9.0

## MySQL

[MySQL Server](https://dev.mysql.com/downloads/mysql/)

## Bibliotecas

### `npm i`

### `npm i --prefix client`

### `npm i --prefix server`

# Executar versão dev

### `npm run both`

# Build

### `npm run build --prefix client`

# Executar versão prod

### `npm run prod`
